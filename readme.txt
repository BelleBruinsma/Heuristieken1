To run any of the search algorithms, open a Search file (eg. Opdracht_A_Search.py) and depth_first.py. Adjust the
parameters of depth_first.py to whatever the top comment says in the Search file. Run the Search file. Output given
at the bottom is the amount of space that has to be filled in the last plate. To visually see the result,
combination_check saves the plates in a Plate object, these object have the property .state, printing that will give
a numpy-matrix based on the results.
